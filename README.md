# Guide de démarrage de l'API

## Installation

```bash
$ npm install
```

## commande pour démarrer l'application

```bash
# development
$ npm run start ou npm start
# Ouvrir le lien http://localhost:3001 dans votre navigateur pour voir l'application.

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
